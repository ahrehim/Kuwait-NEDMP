# Kuwait National Environmental Data Management Plan

Deployment package for the bilingual Kuwait NEDMP website, version 1.9.0.

## GitHub upload

1. Open the `ahrehim/Kuwait-NEDMP` repository.
2. Select **Add file**, then **Upload files**.
3. Drag the contents of this deployment folder into the upload area. Keep `index.html` at the repository root and preserve the `assets` and `documents` folders.
4. Commit the uploaded files to the `main` branch.
5. In **Settings > Pages**, deploy from the `main` branch and the root folder.

Do not upload the 51 MB standalone HTML edition. It is intended for offline use only.
